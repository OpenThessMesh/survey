# Scalability & large-scale methods

Methods addressing reconstruction of large scenes beyond the GPU memory limits of a single model.

**Key open question:** None of these address the feed-forward scalability problem — how to run DUSt3R/VGGT on thousands of images. This is a genuine research gap ThessMesh aims to address.
